Desarrolla Front-end like a boss aprendiendo Boostrap3

En este workshop aprenderás...

En este workshop de 7 horas de duración aprenderás cómo optimizar tu flujo de trabajo a la hora de realizar el front-end de un proyecto web. La metodología apoyada en el framework Boostrap 3 parte desde la fase de diseño hasta finalizar en la fase de desarrollo.

Aprenderás a diseñar para Boostrap, a orientar el diseño bajo la filosofía mobile first responsive web design y descubrirás la metodología adecuada para convertir ese diseño en una maqueta HTML5 de una forma ágil y sencilla.

Como siempre podrás complementar todo lo que aprendas durante el workshop compartiendo experiencias con otros asistentes y el/los ponentes durente la comida o la cena típica asturiana que degustarán todos los que se anime
